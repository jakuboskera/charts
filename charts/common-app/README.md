# common-app

Generic Helm chart for deploying applications. Provides a **Deployment**, **Service**, and an optional **ConfigMap** for environment variables.

## Usage

```bash
helm install my-app charts/common-app \
  --set image.repository=my-service \
  --set image.tag=1.0.0
```
